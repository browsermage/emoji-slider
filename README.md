* Node

<!-- STARTING -->
## Getting started 

1. Clone the repo
   ```sh
   git clone https://github.com/colorsofcode/emoji-slider.git
   ```
2. Move to the new directory
    ```sh
    cd emoji-slider
    ```
3. Start up the development environment
   ```sh
   npm install
   ```
4. Start up the development environment
   ```sh
   npm run dev 
   ```
